# CLAUDE.md

Marketing website for [kylix-pqc](https://github.com/crane-valley/kylix). Built with Astro.

## Commands

- `npm run dev` - Dev server
- `npm run build` - Build (generates OG image)

## Key Files

- `src/pages/index.astro` - Homepage
- `scripts/generate-og-image.mjs` - OG image generator (edit SVG here)
- `public/og-image.png` - Auto-generated, don't edit manually
